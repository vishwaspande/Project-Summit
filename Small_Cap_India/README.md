# 🏦 Super Investor Stock Analyzer — Small Cap India

Interactive dashboard analyzing **360 Indian stocks** held by India's top super investors — Rakesh Jhunjhunwala, Radhakishan Damani, Mohnish Pabrai, Ashish Kacholia, Porinju Veliyath, Dolly Khanna, Vijay Kedia, and more.

## ✨ Features

- **25-metric scoring engine** across Valuation, Quality, Growth, Safety, Momentum, and Conviction
- **Multi-factor composite score** (0–100) for each stock
- **Smart classification** — Deep Value, GARP, High Growth, High Quality, Low Debt, Dividend
- **Interactive table** — sort by any metric, filter by sector/tag, search by stock/investor
- **Detail panel** — radar chart breakdown, 52-week positioning, full metrics
